## Description

A WooCommerce plugin that lets store owners display sleek, real-time toaster popups, instantly notifying customers when a product is added to the cart enhancing clarity, engagement, and purchase confidence.

- [Read Official Documentation](https://www.gethandytools.site/plugin/handy-added-to-cart-toaster-notifier-for-woocommerce/)

## Why You Should Use It?

Explore the comprehensive list of features and compelling reasons to choose Handy Added To Cart Toaster Notifier for your WooCommerce store.

- **Stylish Toaster Modal** - Displays a clean, modern toaster modal every time a product is added to the cart, giving customers clear feedback and improving UX.
- **Customizable Toaster Design** - Customize colors, text, icons, spacing, and shadows to match your store’s branding and create a consistent shopping experience.
- **Adjustable Toaster Position** - Choose where the toaster appears top, bottom, left, right, or corner giving full flexibility in how the message is displayed.
- **Added to Cart Notification** - Shows a confirmation message that reassures customers that their item was successfully added, reducing confusion and misclicks.
- **Product Thumbnail Display** - Includes the product image inside the notification so customers can visually confirm what they just added to the cart.
- **Animated Show & Hide Effects** - Beautify and smooth fade, slide, or bounce animations draw attention naturally without interrupting the browsing flow.
- **Auto-Close Timer Control** - Set how long the notification remains visible auto-hide quickly or stay longer depending on your UX/UI design strategy.
- **Support AJAX Add To Cart** - Compatible with AJAX add-to-cart buttons, ensuring instant notifications without page reloads for better user experience.
- **Mobile Responsive** - The toaster notification adapts beautifully to all screens, making it ideal for mobile-heavy WooCommerce stores.

## Installation

### Manual Installation via FTP
- Download the plugin package to your computer.
- Extract the downloaded file (if it is in .zip format).
- Using your preferred FTP client, upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation.
- Log in to your WordPress dashboard.
- Go to Plugins → Installed Plugins and click Activate next to the plugin name.

### Manual Installation via WordPress Admin (Upload Method)
- Download the plugin archive (.zip file).
- Log in to your WordPress dashboard.
- Navigate to Plugins → Add New.
- Click the Upload Plugin button at the top of the page.
- Select the downloaded .zip file and click Install Now.
- After installation completes, click Activate to enable the plugin.
